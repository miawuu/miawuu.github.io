Event Miawu 1.0

Event yang diselenggarakan pada tanggal 10 Juli 2020 ini merupakan event pertama yang diadakan oleh Tribe Miawu. Tujuan dibuatnya event ini adalah untuk Meningkatkan kesadaran para Pemain TFM akan keberadaan Tribe Miawu, sekaligus bersilaturahmi. Di Event ini ada berbagai macam lomba, antara lain: Survivor, Tanya Jawab, Cover Lagu, Uno, dan Racing.



Event Miawu 2.0

Event yang diselenggarakan pada tanggal 12 Desember 2020 ini merupakan event Kedua yang diadakan oleh Tribe Miawu. Tujuan dibuatnya event ini adalah untuk Meningkatkan kesadaran para Pemain TFM akan keberadaan Tribe Miawu, sekaligus bersilaturahmi. Di Event ini ada berbagai macam lomba, antara lain: Circuit, Parkour, Survivor, Racing, dan Gambar.
